O que é CSS?


O CSS é aquele responsável pela estilização de um elemento como essa seção de texto (caixa azul). Assim, é possível atribuir configurações como cor, tamanho, posicionamento, etc; a um item.

EXEMPLO:


HTML

<section class="texto1">
<h1> O que é CSS </h1>
<p>
O CSS é aquele responsável pela estilização de um elemento como essa seção de texto (caixa azul). Assim, é possível atribuir configurações como cor, tamanho, posicionamento, etc; a um item.
</p>
</section>

CSS

section{
color: aliceblue;
background: linear-gradient(to right, #00276c, #00003d);
margin-top: 5%;
width: 90%;
margin-left: 5%;
border-radius: 20px;
padding-bottom: 2%;

& h1{ padding-top: 2%;
margin-left: 2%;
animation: Transicao 1ms ease-in-out;
animation-timeline: view();
animation-range: cover 0% cover 50%;
}

& p{ margin-top: 2%;
margin-left: 2%;
margin-right: 2%;
animation: Transicao 1ms ease-in-out;
animation-timeline: view();
animation-range: cover 0% cover 50%;
}
}
