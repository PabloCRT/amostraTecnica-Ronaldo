Estrutura HTML


A estrutura HTML como citada antes, é construida por meio de tags, onde cada uma exerce uma função, sendo semântica, única ou para configurar o arquivo. Um bom exemplo é o texto que está sendo lido agora, uma vez que a tag semântica <p> envolve o conteúdo escrito dentro de outra tag semântica <section>.
As tags semânticas existem para organizar o projeto HTML e garantir sua fácil manutenção. Esse tipo de tag não tem uma função unica, no entanto, seu uso é extremamente recomendado, sendo exigido até mesmo no cadastro do projeto no google em um processo chamado de indexação.
