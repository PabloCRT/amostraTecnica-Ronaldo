PROGRAMAÇÃO WEB

O que é HTML?

HTML (HiperText Markup Language) - Nade mais é do que uma linguagem de marcação de hipertexto, ou seja, é a responsável pelo conteúdo "disposto" na página web. Sendo assim, é de boa conduta afirmar que o conteúdo que está sendo lido por você neste exato momento existe graças à mesma linguagem tratada neste trecho.
A linguagem HTML utiliza as tags para definir elementos que podem ou não ser vistos na página web, sendo que, cada tag tem uma função específica, seja como uma função própria na página, semântica do código ou até mesmo configurações do documento HTML.

O que é CSS?

CSS (Cascading Style Sheets) - Se trata de uma lnguagem de estilização, ou seja, é responsável pelo Design da página web. Assim, pode-se observar que as seções de texto que abrangem este conteúdo são uma tag/"elemento" HTML estilizado no CSS, onde configurações como cor, tamanho e forma são atribuídas ao mesmo.

O que é JavaScript?

JS (JavaScript) - Esse sim é uma linguagem de programação. O mesmo, é responsável pelo Back-End (O que está por trás das aplicações) da página web, ou seja, é aquele que trabalha com a parte lógica da aplicação web, sendo responsável por estruturar eventos de interações do usuário com a página.
O Back-End trata de tudo que deve ser calculado após o usuário interagir com algum elemento na página, como por exemplo uma imagem que varia após o usuário interagir com um botão.
